## ⚙️ Requisitos

Antes de executar o jogo, certifique-se de possuir:

- Python 3.10 ou superior
- Pip instalado e configurado
- Terminal compatível com cores ANSI
- Biblioteca `pygame==2.6.1`

> Projeto desenvolvido e testado utilizando Python 3.10+ e pygame 2.6.1.

---

## 📦 Instalação das Dependências

Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/A-Baia-do-Homem-Morto.git
```

Acesse a pasta do projeto:

```bash
cd A-Baia-do-Homem-Morto
```

Instale as dependências necessárias:

```bash
pip install -r requirements.txt
```

---

## ▶️ Executando o Projeto

Após instalar todas as dependências, execute o jogo com:

```bash
python "A Baía do Homem Morto.py"
```

---

## 📁 Dependências Utilizadas

| Biblioteca | Finalidade |
|---|---|
| pygame | Reprodução de música, áudio e efeitos sonoros |
| random | Geração de eventos aleatórios e sistema de combate |
| time | Controle de pausas, ritmo e transições |
| shutil | Centralização dinâmica de elementos no terminal |
| re | Manipulação e limpeza de códigos ANSI |
| os | Gerenciamento de arquivos, diretórios e assets |

> As bibliotecas `random`, `time`, `shutil`, `re` e `os` pertencem à biblioteca padrão do Python, não sendo necessário instalá-las manualmente.
